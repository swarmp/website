## Requirements
- [Node.JS](https://nodejs.org/) `>=14`
- [Yarn](https://classic.yarnpkg.com/en/docs/install)

## Getting started
```sh
yarn          # Install dependencies
yarn dev      # Run development server
yarn generate # Generate the site
```
